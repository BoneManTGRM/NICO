# NICO Platform Architecture

NICO is being built as a repair-first cyber defense platform, not just a scanner.

## Core Engine

Protected foundation:

- Reparodynamics engine
- RYE scoring
- TGRM repair planning
- drift detection
- verification engine
- repair memory
- governance/safety layer
- secret masking
- audit logs
- report engine
