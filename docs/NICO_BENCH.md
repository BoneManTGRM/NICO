# NICO-Bench

NICO-Bench is a planned fixture-based benchmark suite for measuring whether NICO improves local defensive repair workflows over time.

## Planned metrics

- finding quality
- repair plan quality
- verification success
- false-positive control
- secret masking success
- approval boundary success
- swarm risk control
- regression safety
- time-to-repair
- repair memory usefulness

No benchmark result is claimed here. This document defines planned local-only benchmark direction.
