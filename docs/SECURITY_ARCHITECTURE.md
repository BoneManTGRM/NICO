# NICO Security Architecture Foundation

NICO is a defensive-only, repair-first cybersecurity platform foundation.

## Current architecture direction

- Local-first scanning for authorized repositories and fixtures.
- Masked evidence in findings, logs, reports, and audit records.
- Approval-gated repair planning.
- Disabled-by-default connector foundations.
- Tenant isolation and RBAC policy foundations for future SaaS planning.

## Not production ready

This repository does not claim production SaaS readiness, production-grade encryption, hosted auth, billing, or live external scanning. These areas require future hardening, threat modeling, review, and deployment design.
