# Comprehensive Decision-Grade v5 Release Notes

The Comprehensive report now presents technical score, evidence assurance, and delivery authorization as separate dimensions. It corrects scanner-category attribution, surfaces measured architecture hotspots, replaces raw operational errors with bounded disclosures, adds a decision-grade findings register, and produces executable roadmap work packages and machine-readable exports.

This release does not weaken review controls. Reports remain drafts until an authorized human approves the exact immutable package.
