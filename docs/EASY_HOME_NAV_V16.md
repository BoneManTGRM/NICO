# Easy Home Navigation v16

This update makes the simplified NICO paths visible from every page.

## Why

NICO now has an Easy Mode, Start Job wizard, Scanner to Express flow, and full Command Center. Users should not have to remember URLs or understand the entire command center before starting.

## What changed

- Adds a sticky global navigation bar to every page.
- Highlights Easy Mode as the first user path.
- Links directly to Start Job, Scanner to Express, Guided Workflow, and Command Center.
- Adds mobile-responsive wrapping so the navigation remains usable on phones.

## Intended flow

`Easy Mode -> Start Job -> Scanner to Express -> Report review -> Export/signoff`

## Safety rule

The global navigation does not run scans or change code. It only makes the safer workflows easier to find.
