# Easy Mode UI v14

This update adds `/easy`, a simplified landing page for non-technical NICO use.

## Why

The command center is powerful, but it exposes many workflows at once. Easy Mode gives the user four plain choices and a short safe workflow before they see advanced controls.

## New page

- `/easy`

## Main choices

- Check a repo → Start Job.
- Make a client report → Scanner to Express.
- Understand the process → Guided Workflow.
- Use everything → Command Center.

## Simple workflow

1. Start job.
2. Run scanner when possible.
3. Run Express.
4. Review red/yellow items.
5. Export after review.

## Plain rules

- Green does not mean done.
- Unavailable is not passed.
- Use authorized targets only.
- Start simple before using the full command center.

## Related page update

`/start-job` now links back to `/easy` so users can return to the simplest navigation path.
