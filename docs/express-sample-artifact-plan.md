# Express Sample Artifact Plan

This branch tracks the post-release task for the NICO Express Technical Health Assessment sample.

Checkpoint release: v0.1.0-express-assessment
Checkpoint branch: checkpoint/v0.1.0-express-assessment

Goal: generate a downloadable sample output for the Express Pack so the actual markdown and JSON files can be reviewed before customer delivery.

Acceptance checks:

- Standard assessment reports are created.
- Express Pack reports are created under assessment_latest/express_pack.
- Root evidence_manifest.json and Express Pack evidence_manifest.json both exist and are different.
- assessment_latest.json includes reports.express_pack status completed.
- Report language is professional, evidence-bound, and honest about limitations.

Recommended artifact name: nico-express-assessment-sample
