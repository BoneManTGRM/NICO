# Accurate Green Reporting Summary

This change separates the three states that were previously collapsed into one yellow badge:

- technical score and score band;
- evidence assurance;
- canonical risk disposition.

Strong technical scores are no longer visually presented as weak merely because evidence remains review-limited. Review-limited evidence remains visible and continues to block verified-green claims.

The report now rewrites legacy status-colored headings, publishes a verified-green readiness table, and records machine-readable blockers for every control that has not earned green.
