# NICO SaaS Business Model

## Free / Local Edition

- local scans
- local reports
- basic RYE/TGRM
- limited repair memory

## Pro Edition

- GitHub scans
- repair packs
- better reports
- recurring scans
- Codex repair prompts
- AI-agent security checks

## Business / Agency Edition

- multi-project dashboard
- branded reports
- team access
- client portals
- recurring security reports
- approval workflows

## Platform / Marketplace

- build connectors
- build repair packs
- sell templates
- sell compliance packs
- sell industry-specific modules
- platform fee or marketplace percentage

Positioning: NICO is an autonomous security engineer for AI-built apps. It finds drift, ranks repairs, generates fixes, verifies them, and learns what keeps breaking.
