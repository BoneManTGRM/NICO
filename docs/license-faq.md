# NICO License FAQ

This FAQ explains the repository licensing posture in plain language. It is not legal advice and does not replace the full license text.

## Is NICO open source?

No. NICO is source-available, but the default license restricts commercial use.

## What does the default license allow?

It allows non-commercial review, education, testing, internal evaluation, and research use.

## What is not allowed under the default license?

Business use, paid work, customer work, resale, public service offerings, and revenue-generating use require separate written permission.

## When is a commercial license needed?

A commercial license is needed before NICO is used for business operations, customer delivery, paid services, contractor work, or other revenue-generating activity.

## Can NICO be used for customer reports?

Only if the signed commercial scope allows it. The order form should state that right clearly.

## Can NICO be offered as a service?

Only if the signed commercial scope allows it. The order form should state that right clearly.

## Can NICO be modified?

Private changes are allowed only within the applicable license scope. Sharing, selling, or using modified versions commercially requires written permission.

## Do NICO outputs replace human review?

No. NICO outputs are decision-support materials. Human review is still required before important external or production decisions.

## Which files should be reviewed?

Start with:

- `LICENSE`
- `COMMERCIAL_LICENSE.md`
- `docs/commercial-license-order-form.md`
- `docs/commercial-licensing-workflow.md`

Review final customer documents with qualified counsel before use.
