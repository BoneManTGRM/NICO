# Accurate Green Review Checklist

- [ ] Technical score is shown using the score-derived band and color.
- [ ] Evidence assurance is shown separately.
- [ ] Canonical risk disposition is preserved.
- [ ] No missing, failed, timed-out, unavailable, or untriaged evidence is represented as verified green.
- [ ] Every non-green control lists evidence-backed closure requirements.
- [ ] Markdown, HTML, JSON, PDF, and live UI agree.
- [ ] Human approval remains separate from technical scoring.
