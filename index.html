Defensive Architecture Foundations

These cards summarize the new local-only foundations added for the next NICO architecture phase. They are planning and control surfaces, not production SaaS features.

Internal Agent Security

Detects broad permissions, unsafe memory access, missing approval gates, mutation risk, export risk, and connector exposure.

masked findings

Controlled Defensive Swarm

Defines least-privilege roles for Scan, Drift, RYE, TGRM, Verification, Memory, Compliance, Report, Connector Guard, and Supervisor agents.

approval gated

Secret Masking / Vault Foundation

Stores placeholder references, denies unapproved access, masks sensitive values, and avoids raw private-value reporting.

masked only

RBAC / Approval Gates

Adds role-based policy foundations for owner, admin, analyst, developer, and viewer access.

least privilege

Tenant Isolation

Adds tenant context and resource scoping foundations for future hosted planning.

future guard

Connector Guard

Defines disabled-by-default connector policies with allowed scopes, blocked operations, reference requirements, and audit rules.

disabled by default

Sandboxed Scanner Policy

Adds local execution policy foundations with safe working directories, timeouts, masked output, and no network by default.

local only

Cyber Twin

Plans a local model of authorized assets, files, routes, dependencies, findings, repairs, verification history, and audit events.

planned

NICO-Bench

Plans fixture-based benchmarks for finding quality, repair quality, verification success, masking, and approval boundaries.

fixture-based

Quickstart: Local-only NICO workflow

Run NICO locally on authorized repositories or safe demo fixtures. The public page does not execute these commands.

pip install -r requirements.txt python -m nico scan-test-lab python -m nico scan-drift-demo python -m nico report latest python -m nico verify latest python -m nico memory python -m nico policy show python -m nico scanner-availability pytest -q python run_local.py

Frontend local workflow:

cd apps/web npm install npm run lint npm run build npm run dev
